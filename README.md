MiniOS
CSE341 Lab Project (Fall 2025)

1. User Login System (password masking)
	Users will be able to log in with their username and password. Passwords will be masked. (****)

2. File Manager Simulation (create/delete file names)
	Create a file with a specific file. Can add, delete and list files. Has a fixed length file name. 

3. Calculator (advanced arithmetic)
	Addition, Subtraction, Multiplication, Division, modulus and Power

4. Notepad System (store text in array)
	Write any text, edit, delete: (Max: 200 Charactors)

5. Task Manager (view running modules)
	Show Running Programs

6. Command Line Mode
	Simulate simple command line interface with basic commands.

7. History Stack (view previous commands)
	Save executed command and show them highest 20 commands(latest)

8. System Clock: 
	Show system clock latest.
